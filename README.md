# bndwms-flet

Порт клиента **BnDWMS** с PyQt на **Flet**.

## Что реализовано
- Авторизация (ФИО + PIN) как в оригинальном проекте.
- Основные вкладки:
  - Приёмка
  - Перемещение
  - Списание
  - Инвентаризация
  - Остатки
- Настройки подключения (PostgreSQL URL) с локальным сохранением.
- Горячие клавиши: `F1/F2/F3/F4/F5`.
- Тёмная тема с оранжевым акцентом в стиле оригинала и BnDChat-Flet.

## Запуск
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

## Примечание
Это базовый перенос UX и ключевых рабочих экранов на Flet без самовольного расширения функционала.
